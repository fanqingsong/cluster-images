# ImageKmeans
to cluster the images by k-means algorthm.

# run
运行环境：
python3.7

包管理工具：
pipenv
[ik]

运行命令：
```
pipenv install

# 2 means cluster by 2 class
pipenv run ImageKmean.py 2
```

# see the result
in the images folder, cluster-x is the result.
